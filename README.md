# japan-kanko-dashboard
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A dashboard for displaying the number of tourists visiting Japan by prefecture and city.

## Demo
- [Tourist Arrival Dashboard](https://code4fukui.github.io/japan-kanko-dashboard/)
- [Year-on-Year Comparison](https://code4fukui.github.io/japan-kanko-dashboard/compprev.html)

## Features
- Displays tourist arrival data by prefecture and city
- Provides year-on-year comparison of tourist arrivals
- Visualizes the data using treemaps and line charts

## Data / API
- The data is sourced from the [Digital Tourism Statistics Open Data](https://www.nihon-kankou.or.jp/home/jigyou/research/d-toukei/) provided by the Japan Tourism Agency.
- The data is available on [GitHub](https://github.com/code4fukui/japan-kanko-stat) and as a [CSV file](https://code4fukui.github.io/japan-kanko-stat/data/all.csv).
